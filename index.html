CALCULATOR

A Calculator App created with HTML, CSS, Bootstrap, and JavaScript that performs arithmetic calculations (such as addition, subtraction, multiplication, and division) and displays the results interactively.
HTML: Contains the structure of the calculator with an input field to display the numbers and results.
CSS: CSS is used to style the calculator's appearance, including button sizes, colors, font sizes, and overall layout. With Bootstrap (a front-end framework), additional styling for grid systems, responsiveness
JavaScript: JavaScript handles the calculation logic and button functionality, managing user input, performing operations, and displaying the result.
